# **포트폴리오 웹사이트**

## 📌 **Summary**

**포트폴리오 용도로 제작한 웹사이트**입니다.

Flutter 학습겸 포트폴리오 제작겸 개발하였습니다.

### * **주요 기능**

- 간단한 자기소개
- 인적 사항
- 기술 스택
- GitHub 및 BOJ 링크
- 프로젝트 경험
- 업무 경력

## 🤔 **Background**

포트폴리오 같은 스크롤이 연속적으로 이어지는 부류의 웹페이지가 Sliver를 이해하기 좋을것 같아서 개발하게 되었습니다.

## 🔨 **Technology Stack(s)**

- Frontend: Flutter,
- Deployment: GitHub workflow
